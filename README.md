# 🎬 Filmogram
Filmogram est une application web de gestion de catalogue cinématographique développée dans le cadre de l'épreuve E4/E5 du **BTS SIO option SLAM**. Elle propose une interface utilisateur pour consulter des films et un panel d'administration sécurisé pour gérer le contenu.

## 🚀 Fonctionnalités Clés
- **Catalogue Utilisateur** : Affichage dynamique des films, descriptions, note d'âge et genres.
- **Panel d'Administration (Thème Sombre Premium)** :
  - **Authentification forte** : Double sécurité (mot de passe + question secrète).
  - **Gestion des films (CRUD)** : Ajout, modification, suppression et affichage des films.
  - **Gestion des utilisateurs** : Administration des comptes (ajout, suppression).
- **Sécurité intégrée** : Requêtes préparées PDO (anti-injection SQL), hachage des mots de passe, contrôle d'accès par sessions.

## 🛠️ Stack Technique
- **Backend** : PHP 8, appels API internes
- **Frontend** : HTML5, CSS3 Vanilla, JavaScript
- **Base de données** : MySQL
- **Serveur local** : Apache (via XAMPP)
